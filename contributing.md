### Detailed Installation Guide (June 2026 Update)

#### Step 1: Download & Extract
- Click the download button above
- Download the latest release (beammp-server-download)
- Extract the archive using WinRAR or 7-Zip
- Package size: ~15 MB (server only)

#### Step 2: Configure the Server
- Open `ServerConfig.toml` in a text editor
- Set server name, description, max players
- Set a password (optional)
- Configure port (default 30814)
- Add your BeamNG mods to the `Resources` folder

#### Step 3: Run the Server
- Right-click `BeamMP_Server.exe`
- Select **"Run as Administrator"** (for port access)
- The console will show "Server started"
- Share your IP and port with friends
- Players join via BeamNG multiplayer menu

**Done! Your BeamMP server is running — zero cost.**
